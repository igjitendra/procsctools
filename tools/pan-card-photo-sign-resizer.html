NSDL vs UTI Specifications Chart

NSDL portal par photo aur signature documents apply karne ke limits and values is tarah hain:

  • NSDL Photo: Size matches 3.5 cm x 2.5 cm. Pixel resolution exactly 276 x 197 px at 200 DPI limit, size must be under 20KB.
  • NSDL Signature: Size matches 4.5 cm x 2.0 cm. Pixel resolution exactly 354 x 157 px at 200 DPI, size must be under 10KB.

Wahin dusri taraf UTIITSL portal details thodi alag demand karti hai:

  • UTI Photo: Size matches exactly 213 x 213 px square, color format matching 300 DPI limits under 30KB with white background.
  • UTI Signature: Size matches 400 x 200 px (2:1 aspect ratio), at 600 DPI limits under 60KB.

Frequently Asked Questions

Get answers to common PAN document sizing questions

What is the DPI requirement for PAN Card photos?
NSDL requires photos scanned at 200 DPI (dots per inch), while UTIITSL demands 300 DPI for photos and 600 DPI for signatures. Our tool automatically adjusts pixel mappings and compression output formats to comply with these rules.
Why does UTIITSL signature require 600 DPI?
High DPI values prevent blurring in print audits. When choosing the 'UTI Signature' preset, our resizer outputs a clean high-definition 400x200 pixel canvas compressed to stay under the 60KB maximum boundary.
How does the white background check work?
Often scanned signatures have greyish background shadows. The "Auto White Background" check uses HTML canvas composition to fill empty and transparent segments with pure white, ensuring clear, high-contrast signature results.
Is this tool completely free for CSC VLEs?
Yes, Pro CSC Tools provides this resizer completely free of cost. There are no watermarks, registration gates, or limits on how many documents you can process daily.
What if the downloaded image size is still slightly above 20KB?
Our smart script automatically runs a compression loop to adjust the file weight below the threshold (e.g. 20KB/10KB) during download. Make sure to use high-contrast images for optimal results.
