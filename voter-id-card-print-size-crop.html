Voter ID Card Print Ke Liye Correct Size Aur Crop Kaise Karein

e-EPIC PDF se PVC voter ID print - free browser tool se.

EPIC (Electoral Photo Identity Card) India ka primary voter ID. e-EPIC (2021 launched) ko ghar par PVC card ki tarah print kar sakte hain. Complete process yahan.

EPIC Correct Size

ParameterValue
Standard85 x 54 mm
Paper4x6 glossy
LayoutFront + back

Step 1: e-EPIC Download

  1. voters.eci.gov.in
  2. "Download e-EPIC"
  3. EPIC number/reference number
  4. Mobile OTP verify
  5. PDF download

Step 2: 4x6 Crop

  1. Aadhar Crop Tool
  2. "Voter Card" icon select
  3. PDF upload (password nahi chahiye)
  4. 4x6 Print → PROCESS

Step 3: PVC Print

Image ko Windows Photos me open, 4x6 glossy paper par print, PVC pouch me laminate, cutter. Full: PVC print process.

Multiple Voter IDs Ek A4 Me

Poore parivaar ke liye tool me A4 Print select — ek A4 me 2 EPIC (front + back) fit.

Tip: Voter ID me QR code poora dikhna chahiye — crop me include karo warna authentication problem.
