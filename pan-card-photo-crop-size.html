PAN Card Photo Crop Size Kya Hota Hai?

NSDL/UTI e-PAN se PVC print-ready — browser me free.

PAN card ka standard size credit card jaisa — 85.6 x 54 mm (3.38 x 2.13 inch). Correct crop hi PVC print ka pehla step. NSDL aur UTI e-PAN dono ka process yahan.

PAN Correct Size

ParameterValue
Physical85.6 x 54 mm
Aspect ratio1.586:1
Print DPI300+ recommended
Paper4x6 glossy

NSDL e-PAN Crop Steps

  1. Aadhar Crop Tool → "PAN Card (NSDL ONLY)"
  2. PDF upload
  3. Password: DOB DDMMYYYY format (example: 15051998)
  4. 4x6 Print → PROCESS
  5. Windows Photos me print

UTI e-PAN Note

UTI layout different hai — tool NSDL ke liye optimized. UTI ke liye pehle PDF ko image me convert kar ke Photo option use karein.

Clarity Rules

  • Print DPI 300+ (kam par blur)
  • Signature area sharp — IT verify me use
  • QR code area crop me include — modern PAN mandatory

Common Mistakes

  • Password DDMMYYYY, YYYYMMDD nahi
  • Screenshot use → quality drop, PDF direct upload karein
  • Manual crop → signature/QR cut, tool auto safe
Tip: PAN PVC ka process Aadhaar jaisa — PVC guide.
